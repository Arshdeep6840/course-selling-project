# Course Selling Project

This is a web application built with Python and Flask for selling online courses. The application allows users to browse courses, register for an account, enroll in courses, and view course content.

## Features

*   **User Authentication:** Users can register for a new account, log in, and log out.
*   **Course Catalog:** A list of all available courses is displayed on the main page.
*   **Course Details:** Each course has a dedicated page with a description, price, and image.
*   **Course Enrollment:** Authenticated users can enroll in courses.
*   **Video Content:** Enrolled users can access and watch the course videos.
*   **Database Seeding:** The project includes a script to seed the database with sample courses and videos.

## Technologies Used

*   **Backend:** Python, Flask
*   **Database:** SQLite
*   **ORM:** SQLAlchemy
*   **Database Migrations:** Alembic
*   **Frontend:** HTML, CSS, Bootstrap
